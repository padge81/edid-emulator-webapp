Below is a single script you run once:
EnableBluetoothPAN.sh

It will:

✅ Install required packages
✅ Enable Bluetooth PAN (NAP mode)
✅ Create a persistent systemd service
✅ Assign a static IP to the PAN interface
✅ Start automatically on boot
✅ Work with Android & iOS
✅ Require no changes to your Flask app

✅ EnableBluetoothPAN.sh (FULL SCRIPT)

Run once:

chmod +x EnableBluetoothPAN.sh
